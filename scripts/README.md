# News Sentiment Analysis & Stock Prediction

## Project Overview
This project analyzes the relationship between news sentiment and stock price movements to build a predictive trading signal system.

## Tasks Completed

### Task 1: Data Exploration (EDA)
- Loaded and cleaned news dataset
- Performed exploratory data analysis
- Identified publication trends and publisher patterns

### Task 2: Technical Indicators
- Implemented financial indicators:
  - Moving Averages (SMA/EMA)
  - RSI (Relative Strength Index)
  - MACD (Moving Average Convergence Divergence)
- Visualized stock trends and momentum signals

## Tech Stack
- Python
- Pandas
- Matplotlib
- TA-Lib / custom indicators

## Next Steps
